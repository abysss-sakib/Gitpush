# ✨ Overview

**GitPush** is a modern GitHub repository manager designed to simplify repository file management through a polished web interface.

It allows users to:
- browse repository files
- upload folders
- replace files
- delete content
- manage repositories visually

without using Git commands manually.

Built with a strong focus on:
- ⚡ Performance
- 🎨 Clean UI
- 📁 File management simplicity
- 🚀 Smooth workflow

---

# 🔥 Features

- 📂 Modern file explorer UI
- 📤 Upload files & folders
- 🔄 Replace existing files
- ❌ Delete single or multiple files
- ⚡ Atomic GitHub commits
- 📱 Responsive design
- 🌙 Light & dark mode
- 🧠 Smart upload handling
- 🚀 Optimized for large files
- 🔐 GitHub token verification
